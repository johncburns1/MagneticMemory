Magnetic Core Memory




Magnetic Core Memory and Whirlwind (1953)

  • First reliable high-speed RAM for computers
  • Invented June 13th 1949 by Jay Forrester
  • Widely used for main memory up until the mid-1970s
  • Process:
    • Uses tiny magnetic toroids (rings)
    • Wires are threaded through the rings (cores) to read and write information to them
    • Each toroid represents one bit of memory
    • Cores can be magnetized in 2 different ways --> corresponds to their value
    • Smaller the cores and wire, greater the memory density of the core
  • Whirlwind (1953, MIT) --> First computer to utilize magnetic core memory

"It took about seven years to convince people in the industry that magnetic core memory would work, and it took the next seven years to convince them that they had not all thought of it first."

-Jay Forrester
